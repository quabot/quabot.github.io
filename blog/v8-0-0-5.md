Added & Changed

- Added documentation links to some modules that didn't have them.
- Updated the premium page design with new buttons & feature tables.
- Added a check for embed total character count. If exceeds 6000 (limit) editing becomes unavailable.
- Auto Purge & Auto Delete modules now have 3 available channels for free and up to 50 for premium.
- Reaction roles now have 5 free messages and 50 with premium.

Fixed

- Fixed fonts not working for welcome cards.
- Fixed options for selections having incorrect colors on some webbrowers.
- Fixed various icon styling & sizing issues around the site.
- Fixed the about me edit tooltip & name being the banner text.
- Fixed commands overflow scrollbars on some webbrowsers.
- Fixed some dashboard tabs being available without selecting a server on mobile.
- Fixed the pages not scrolling back up when navigating around the site.
- Fixed message sender not working when the message had fields.
- Various UI fixes.
