---
title: V8.0.2.8
date: 2026-01-11T14:48:00
---

- Fixed an issue where servers without QuaBot would display 'Configure' and allow full dashboard management when this wasn't applicable.
- Fixed an issue where the default settings for the welcome configuration had some incorrectly enabled values.
- Fixed an issue where disabling the welcome/leave message would prevent saving.
- Updated the wording for the permissions notice for role multi pickers & added a wiki link for additional context.
- Added a 'no reaction roles' message when you have no reaction roles.
- Fixed a crash in edge cases on the tickets page.
- Fixed an issue where creating ticket panels was not possible if you deleted one in the past.
- Removed the @ everyone role from the 'ticket open roles' and 'ticket close roles'.
- Fixed having the same role in: allowed/blocked roles, close roles add/close roles remove and open roles add/remove for ticket panels.
- Fixed verification roles not being both in roles to add and roles to remove at once.
- Added the ability to sort reaction role role options.
- Separated the join/leave channels, you can now send them in separate channels.
